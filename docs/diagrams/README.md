# Diagrams

This directory contains diagrams and visual documentation for the SalesIQ project.

## Purpose

Store architecture diagrams, flowcharts, and other visual representations of:
- System architecture
- Data flows
- Component relationships
- Process workflows
- Business logic diagrams

## Supported Formats

- Mermaid diagrams (`.mmd`)
- PlantUML diagrams (`.puml`)
- Draw.io diagrams (`.drawio`)
- Image exports (`.png`, `.svg`, `.jpg`)
- Other diagram formats as needed
